# HachimiMT — Colab

Notebook chạy app dịch Trung→Việt **HachimiMT / MoxhiMT** trên Google Colab
(tận dụng CPU/GPU miễn phí). Bấm nút dưới → `Runtime → Run all` → mở link
`*.gradio.live` để dịch.
